## 使用
发送消息
```
curl -X GET http://<your_url>/api/notify?u=123&m=HelloWorld&t=developmenttoken
```

路由参数说明

| url 路径   | 参数（类型, 说明）        | 说明         |
| ---------- | ---------------------- | ------------ |
| api/notify | m(消息内容)，u(telegram用户id)，t(`api_token`内容) | 发送文字消息 |
